 1. Create OpenRDF Sesame work directory
------------------------------------------
# mkdir /opt/_workapps/rinfo/aduna


 2. Tomcat installation
------------------------------------------
Download core-distribution of Tomcat 6 from http://tomcat.apache.org/download-60.cgi
# cd /usr/share
# tar -xzvf apache-tomcat-6.0.16.tar.gz
# mv apache-tomcat-6.0.16 tomcat6


 3. Tomcat configuration - to run Tomcat as service 
------------------------------------------
[RED HAT LINUX CONFIGURATION]
------------------------------------------

# cp tomcat /etc/init.d
# cp setenv.sh /usr/share/tomcat6/bin
# chmod 755 /etc/init.d/tomcat
# chmod 644 /usr/share/tomcat6/bin/setenv.sh
# adduser tomcat
# service tomcat start
# chkconfig --add tomcat
# chkconfig --level 345 tomcat on
