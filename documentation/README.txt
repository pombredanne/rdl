########################################################################
# Dokumentation
########################################################################

Innehåll
========================================================================

index.html
    Översikt över extern dokumentation, grund för vad som publiceras
    på dev.lagrummet.se.

introduktion/
    Fristående dokument som riktar sig till informationsansvariga och
    teknikansvariga på myndigheter som ska leverera rättsinformation,
    samt till de som vill vidareutnyttja materialet i systemet.

handbok/
    Detaljerade beskrivningar av hur enskilda dokumenttyper publiceras
    i systemet, med exempel.

exempel/
    Exempel på dataformat och användning av standarder riktat till deltagande
    organisationer. Vissa delar av materialet används för att testa systemet.

system/
    Dokumentation om systemet (översiktsbilder mm).

spec/
    Dokumentation av acceptanstester och krav.



Se även
========================================================================
FST - föreskriftshantering som tjänst, en slutanvändartjänst för
myndigheter som vill publicera sina föreskrifter till rättsdatalagret
utan att integrera sina egna system: https://github.com/rinfo/fst/
